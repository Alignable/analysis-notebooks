# private/

Scratch space for files you want on disk but not in git — exported CSVs, intermediate markdown, sample payloads, anything you don't want to think about pushing.

Everything under this directory is gitignored except this README. Organize it however you like; a common pattern is `private/<notebook-name>/`.

Don't put anything here that other contributors will need — if it's worth sharing, commit it to the relevant `notebooks/<user>/` directory instead.
